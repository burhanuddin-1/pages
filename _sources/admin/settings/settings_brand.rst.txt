.. _settings brand:

==========
``brand:``
==========

.. code:: yaml

   brand:
     issue_url: https://github.com/burhanuddin-1/s/issues
     docs_url: https://infinity-search.onrender.com
     public_instances: https://infinity-search.onrender.com
     wiki_url: https://github.com/burhanuddin-1/s/wiki

``issue_url`` :
  URL for issue tracking (e.g., GitHub issues).

``docs_url`` :
  URL for the documentation.

``public_instances`` :
  URL for public instances information.

``wiki_url`` :
  Link to the project wiki (or set to ``false`` to disable)
