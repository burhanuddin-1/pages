utils/searxng.sh¶

To simplify the installation and maintenance of an Infinity Search instance you can use the script git://utils/infinity-search.sh.

Install¶

In most cases you will install Infinity Search simply by running the command:

sudo -H ./utils/infinity-search.sh install all

The installation is described in chapter Step by step installation.

Command Help¶

The --help output of the script is largely self-explanatory:

System Message: ERROR/6 (/mnt/d/code/s/docs/utils/searxng.sh.rst, line 42)

Command ['../utils/infinity-search.sh', '--help'] failed: [Errno 2] No such file or directory: '../utils/infinity-search.sh'
